### 3. 数据入库方式

#### 方式一：API接口入库

**项目创建API**（`app/api/routes/projects.py`）：
```python
@router.post("/", response_model=ProjectResponse)
async def create_project(
    project_data: ProjectCreate,
    db: AsyncSession = Depends(get_db),
    current_user: User = Depends(get_current_user),
):
    """创建新项目"""
    service = ProjectService(db)
    project = DrugProject(**project_data.dict(), created_by=current_user.id)
    created_project = await service.create(project)
    return ProjectResponse.from_orm(created_project)
```

#### 方式二：Excel批量导入

**Excel导入功能**（`app/services/import_export_service.py`）：
```python
async def import_from_excel(self, file: UploadFile, mode: str = "append") -> dict:
    """从Excel文件导入项目数据"""
    # 读取Excel文件
    df = pd.read_excel(file.file)
    
    # 数据验证和转换
    projects = []
    for _, row in df.iterrows():
        project_data = self._validate_project_data(row)
        project = DrugProject(**project_data)
        projects.append(project)
    
    # 批量保存
    for project in projects:
        self.db.add(project)
    
    await self.db.commit()
    return {"imported_count": len(projects)}
```

### 4. 数据验证机制

**数据验证服务**（`app/services/project_service.py`）：
```python
async def create(self, project: DrugProject) -> DrugProject:
    """创建项目（包含数据验证）"""
    # 数据验证
    if not project.project_name or len(project.project_name.strip()) == 0:
        raise ValueError("项目名称不能为空")
    
    # 保存到数据库
    self.db.add(project)
    await self.db.commit()
    await self.db.refresh(project)
    return project
```

### 2. 数据导出功能

**Excel导出**（`app/services/import_export_service.py`）：
```python
async def export_to_excel(self, filters: dict) -> bytes:
    """导出项目数据到Excel"""
    # 查询数据
    projects = await self.query_projects(filters)
    
    # 转换为DataFrame
    df = pd.DataFrame([project.to_dict() for project in projects])
    
    # 生成Excel文件
    output = io.BytesIO()
    with pd.ExcelWriter(output, engine='openpyxl') as writer:
        df.to_excel(writer, sheet_name='项目数据', index=False)
    
    return output.getvalue()
```
