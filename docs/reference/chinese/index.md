## 使用指南

### 如何阅读中文文档

#### 1. 技术术语对照
中文文档中的技术术语会提供英文对照，格式如下：
- **知识库 (Knowledge Base)**
- **检索器 (Retriever)**
- **向量嵌入 (Vector Embedding)**

#### 2. 代码示例
代码示例使用中文注释，便于理解：
```python
# 创建知识库
async def create_knowledge_base(name: str, tenant_id: str):
    """
    创建新的知识库
    
    Args:
        name: 知识库名称
        tenant_id: 租户ID
    
    Returns:
        KnowledgeBase: 创建的知识库对象
    """
    kb = KnowledgeBase(name=name, tenant_id=tenant_id)
    return await kb.save()
```

#### 3. 配置说明
配置参数使用中文说明：
```yaml
# 数据库配置
database:
  host: localhost        # 数据库主机地址
  port: 5432            # 数据库端口
  name: kb              # 数据库名称
  
# LLM 配置
llm:
  provider: ollama      # LLM 提供商
  model: qwen2.5:14b    # 使用的模型
  temperature: 0.7      # 生成温度参数
```
