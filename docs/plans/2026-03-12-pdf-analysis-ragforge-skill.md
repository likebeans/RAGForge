# PDF Analysis RagForge Skill Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the current pdf-analysis skill's pipeline-based flow with direct RagForge orchestration that extracts structured fields to Excel and ingests PDF content into the 项目管理 knowledge base using Markdown chunking.

**Architecture:** Keep the skill thin. The OpenClaw skill will call a local Python CLI that talks directly to RagForge on port 8020 with an API key. The CLI will ensure the target knowledge base uses Markdown chunking, upload the PDF so RagForge/MinerU performs PDF-to-Markdown ingestion, then run schema extraction and save an Excel export locally.

**Tech Stack:** OpenClaw skill markdown, Python 3 standard library HTTP client, RagForge REST API, shell setup script, unittest for helper-level regression checks.
